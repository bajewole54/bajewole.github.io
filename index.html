01
Embedded MLTensorFlow LiteRaspberry PiENGR-222

Real-time fall detection on a Raspberry Pi

In plain termsA small computer that watches motion data and instantly recognizes when someone falls, much like a smoke detector but for falls.

Trained a CNN to detect falls and classify activity from motion-sensor data. The full model was too heavy for real-time alerts on a Pi, so Knowledge Distillation compressed it 63% (3.29 MB to 1.2 MB), latency dropped from 30 to 50 ms down to 10 to 20 ms, and accuracy actually improved to 93.75%. Deployed with TensorFlow Lite, firing LED-matrix alerts in real time.

Motion sensors→ CNN (teacher)→ Distilled model (student)→ TFLite on Raspberry Pi→ LED-matrix alert

Machine learningMLOpsDockerREST APIENGR-222

Income-prediction ML microservice, a Dockerized REST API

In plain termsA machine learning model turned into a real web service: upload a spreadsheet, get predictions back in seconds, packaged in Docker so it runs identically on any computer.

Trained a Logistic Regression classifier on the UCI Adult Census dataset inside a scikit-learn pipeline, tuned with GridSearchCV to roughly 85% test accuracy. Wrapped the model in a Flask and Connexion REST API documented with OpenAPI and Swagger, with endpoints for model info, batch predictions from uploaded CSVs, accuracy evaluation, and on-the-fly visualizations, then containerized the entire service with Docker.
